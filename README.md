# apollo-graphql-react-redux-typescript-sql
Clean boilerplate for React graphQL + SQL with a lot of fancy friends!

- Typescript
- Apollo GraphQL Client
- React
- Redux


## Development
Pre
```
$ npm install
```
Run project
```
$ npm start // run graphql server
$ npm run hot // run client
```
